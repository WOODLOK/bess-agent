# 关键政策时间线索引 (2018–2025)

本文档是 BESS 供应链相关的主要经济体政策与法规的关键节点时间线，供 `tool-policy/SKILL.md` 在阶段三检索时作为快速对照参考。

> 每个条目标注了公布日（announcement）与生效日（effective date）。政策的影响通常在公布日就已开始（市场预期反应），而非等到生效日才显现。

---

## 一、美国 301 条款关税（USTR Section 301）

| 日期 | 事件 | 对 BESS 供应链的影响 |
|:---|:---|:---|

| 2018-08-23 | 第二轮 301 关税（List 2）生效，25% | 进一步覆盖工业零部件 |
| 2018-09-24 | 第三轮 301 关税（List 3）生效，10%（2019-05-10 升至 25%） | 开始覆盖部分电池相关品类 |

| **2024-05-14** | **USTR 宣布储能电池 301 关税从 7.5% 提至 25%，2026 年执行** | 直接影响中国产储能电池的进口成本 |
| 2026 (TBD) | 储能电池 25% 关税正式生效 | |

## 二、通胀削减法案（IRA）与外国敏感实体（FEOC）

| 日期 | 事件 | 对 BESS 供应链的影响 |
|:---|:---|:---|
| **2022-08-16** | **IRA 签署生效** | Section 45X 引入先进制造生产抵免，Section 48 引入投资税收抵免（ITC），但两者均与 FEOC 挂钩——若电池组件或关键矿物来自 FEOC，则无法享受全额补贴 |
| 2023-03-31 | 财政部发布 IRA 电池采购要求拟议规则（NPRM） | 明确了关键矿物与电池组件的 FEOC 排除规则框架 |
| 2023-12-04 | FEOC 拟议规则发布 | 定义"外国敏感实体"的范围（包括中国、俄罗斯、朝鲜、伊朗政府控制的企业） |
| 2024-05-03 | FEOC 最终规则发布 | 最终确认 FEOC 定义与合规时间表 |
| 2024-06 | 关键矿物 FEOC 排除开始适用于部分税收抵免 | |
| 2025-01 | 电池组件 FEOC 排除开始适用 | |

## 三、维吾尔强迫劳动预防法（UFLPA）

| 日期 | 事件 | 对 BESS 供应链的影响 |
|:---|:---|:---|
| 2021-12-23 | UFLPA 签署 | 建立"可反驳推定"（rebuttable presumption）机制——任何全部或部分在新疆生产或与新疆相关的货物被推定为以强迫劳动生产，禁止进口，除非进口商能提供"清晰且令人信服的证据"反驳 |
| **2022-06-21** | **UFLPA 正式生效** | CBP 开始执法，首批执法对象为太阳能光伏多晶硅，随后扩展至其他行业 |
| 2022 Q3–Q4 | CBP 扣留范围扩展到电池供应链相关品类 | 进口商开始主动加强对电芯、铝制外壳、铜箔、电解液溶剂的原产地追溯 |
| 2023–2025 | 执法持续升级 | 进口商被要求建立从矿山到工厂的完整可追溯文档体系 |

## 四、欧盟电池法规（EU Battery Regulation 2023/1542）

| 日期 | 事件 | 对 BESS 供应链的影响 |
|:---|:---|:---|
| 2020-12-10 | 欧盟委员会发布电池法规提案 | 替代旧电池指令（2006/66/EC），引入碳足迹、回收成分、供应链尽职调查等新要求 |
| **2023-07-28** | **EU Battery Regulation (2023/1542) 正式通过** | |
| 2023-08-17 | 法规在 Official Journal 发布，20 天后生效 | |
| 2024-02-18 | 法规全面生效 | 碳足迹声明要求开始分阶段强制实施 |
| 2025-02-18 | 电动汽车电池碳足迹声明强制 | |
| 2025-08-18 | 供应链尽职调查义务开始适用于经济运营商 | 企业需建立电池供应链的尽职调查政策并接受第三方审计 |
| 2026-02-18 | 工业电池（含储能电池）碳足迹声明强制 | 直接适用 BESS 行业 |
| **2027-02-18** | **电池护照（Battery Passport）要求强制生效** | 所有在欧盟市场销售的工业电池须具备带唯一标识符的数字电池护照，包含碳足迹、回收成分、供应链尽职调查信息 |

## 五、中国石墨出口管制

| 日期 | 事件 | 对 BESS 供应链的影响 |
|:---|:---|:---|
| **2023-10-20** | **商务部与海关总署宣布对石墨实施出口管制** | 天然鳞片石墨及制品（含球形石墨）被纳入出口管制清单，需申请出口许可证 |

| 2024 Q1 | 石墨价格出现上涨（管制效应） | 天然球形石墨 FOB 价从管制前约 $3,650/t 升至管制后约 $3,800-3,950/t |

## 使用说明

在阶段三执行 `tool-policy` 检索时：
1. 先从本时间线中查找事件时间窗口 ±6 个月内的所有政策节点
2. 将匹配到的政策节点作为 WebSearch 的重点检索方向
3. 检索时使用本时间线中标注的精确日期（公布日/生效日）来构造 `time_filter`
4. 如果 WebSearch 返回政策的生效日期与本时间线有冲突，以官方公告日期为准，并在 evidence_item 中标注差异

本时间线保持年度更新。最后更新：2026-05。
