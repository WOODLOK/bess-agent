# 示例输入

本目录包含一份模拟的私域供应链中断材料，供新用户快速体验智能体的完整五阶段分析流程。

## 使用方法

1. 在 Claude Code 中打开本仓库并提及 `@AGENT.md`
2. 当智能体询问来意时，回复"分析这次中断"
3. 智能体将读取 `narrative.md` 作为输入材料，执行完整的五阶段分析

## 案例概要

- **虚构企业**：一家美国加州 BESS 项目开发商
- **中断事件**：2022 年 3-5 月，一批从上海港发往洛杉矶的 280Ah LFP 电芯因上海封控大幅延误
- **涉及因素**：物流（港口拥堵）、原材料（碳酸锂价格冲顶）、合规（UFLPA 清关审查）
- **材料类型**：口述复盘记录 + 三封邮件往来

## 预期产出

智能体完成分析后，应在 `memory/cases/` 下生成一条新的案例条目，包含：
- 因果链分析（识别上海封控为根因，碳酸锂价格为次级因素）
- 延误天数区间估计
- 完整的证据检索日志
- 一条指向本例的 `INDEX.md` 索引条目

## 虚构声明

本示例中的所有企业名称、供应商名称、人名、合同编号、邮件内容均为虚构。但底层时间线（2022 年 3-5 月上海封控）、宏观价格数据（碳酸锂冲顶）和航运指数（SCFI 高位）均基于真实历史事件。
