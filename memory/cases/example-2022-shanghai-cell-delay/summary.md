# example-2022-shanghai-cell-delay

**一句话因果总结**：2022 年 3-5 月上海封控导致港口作业停滞，一批 280Ah LFP 电芯从上海至洛杉矶的海运延误约 60-75 天。同期碳酸锂价格从合同签订时的约 22 万元/吨暴涨至约 50 万元/吨，加大了供应商履约成本压力。清关阶段因部分原材料涉及新疆地区触发 UFLPA 补充审查（额外约 1 周）。项目 COD 推迟约一个季度。

**主要影响因素**：物流, 原材料, 合规与政策

**事件类型标签**：regional_control (covid_lockdown), route_disruption (port_congestion), material_price_shock (lithium_carbonate_surge), compliance_seizure (uflpa_review)

**时间窗口**：2022-02-28 至 2022-06-10

**整体置信度**：medium
